# 🤖 TeraBox Telegram Downloader Bot

A simple Telegram bot that downloads files from public links and sends them directly to users.

## 🚀 Features
- Direct link download
- Automatic file sending
- No commands required
- Works in private chat

## 🛠 Requirements
- Python 3.9+
- python-telegram-bot
- requests

## 📦 Installation

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
pip install -r requirements.txt
